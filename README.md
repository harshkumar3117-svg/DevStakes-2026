# DevStakes-2026# 🎬 CineStream —  Movie Discovery Platform

## 🌌 Overview
CineStream is a modern movie discovery platform designed to deliver a personalized cinematic experience.  
With a sleek UI and  recommendations, users can explore, track, and engage with movies effortlessly.

---

## ✨ Features

### 🔍 Smart Discovery
-  movie recommendations  
- Personalized suggestions based on user preferences  
- Featured spotlight section for immersive browsing  

### 🎭 Genre-Based Browsing
- Browse movies by categories:
  - Action, Comedy, Horror, Sci-Fi, Romance, Thriller, Animation, etc.  
- Smooth horizontal scrolling sections (Trending, Popular)

### ⭐ Ratings & Reviews
- View ratings and movie details  
- Read community reviews  
- Post your own reviews  

### 📄 Movie Details Page
- Detailed information:
- Rating, duration, language, votes  
- Watchlist and sharing options  
- Clean and immersive UI

### ▶️ Watch Now (New Feature)
- Users can click the “Watch Now” button to instantly start streaming the selected movie
- The movie is fetched and played directly from a connected movie server or streaming API
- Provides a seamless viewing experience without leaving the platform
- Supports fullscreen playback and basic controls (play, pause, volume, seek)

### ❤️ Watchlist System
- Add movies to your personal watchlist  
- Track what to watch later  

### 🔐 Authentication System
- Secure login & signup  
- Social login (Google, Apple)  
- Modal-based authentication UI  

### 🎨 Modern UI/UX
- Dark theme with neon gradients  
- Smooth animations and transitions  
- Responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** React 
- **Backend: **Spring
- **API:** Movie data (e.g., TMDB API)  
- **Authentication:** Email 
- **Design:** Gradient + Neon UI  
---

## ⚙️ How It Works

1. Open the CineStream platform  
2. Browse or search for movies  
3. Explore Movie recommendations  
4. View movie details and reviews  
5. Add movies to your watchlist
